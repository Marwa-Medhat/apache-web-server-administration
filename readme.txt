This Project for Apache web server administration
you can 
1-Install / remove web server
	2-Administrate virtualhost /etc/apache2/sites-enabled
		list all virtual hosts, add, delete, disable, enable
	3-Configure authentiation for each virtualhost in .htaccess
		enable auth for a certain virtualhost
		disable auth for a certain virtualhost

There is Menu write number of operation you want
    num 1 to install apache web server
    num 2 to remove apache web server
	num 3 list all virtual hosts
	num 4 add virtual host
    num 5 delete virtual host
	num 6 disable virtual host
	num 7 enable virtual host 
	num 8 enable  auth virtual host 
	num 9 disable auth virtual host 
	num 10 start apache		
	num 11 stop Apache
	num 12 Quit
				
If You write any number not included from 1 to 12 will appear on screen  not valid    

To run project you need to 
1-chmod +x main-option.sh
2-./main-option.sh
